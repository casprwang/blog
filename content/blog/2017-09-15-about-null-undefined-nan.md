## Expressions

```js
null + "";
//=> "null"

undefined + "";
//=> "undefined"

// NaN -> NaN
NaN + "";
//=> "NaN"

NaN + 1;
//=> NaN

NaN + 3;
//=> NaN

// null + int -> int
null + 3;
//=> 3

// NaN + undefined||null -> NaN
null + NaN;
//=> NaN

undefined + NaN;
//=> NaN

undefined + 3;
//=> NaN

// WTF
null + undefined;
//=> NaN
```
